# HU3: Visualizar y buscar usuarios existentes (Frontend)

**Como** Administrador  
**Quiero** ver una lista de todos los usuarios y poder buscar/filtrar por nombre, email, rol o estado  
**Para** gestionar y localizar fácilmente a los empleados.

## Criterios de Aceptación
- La sección “User Management” muestra una tabla/lista con los siguientes datos de cada usuario:
	- Avatar - defecto un icono user
	- Nombre
	- Email
	- Rol
	- Estado (Active, Inactive, Pending)
	- Acciones (Editar, Desactivar)
- Hay un campo de búsqueda para filtrar por nombre o email.
- Hay filtros tipo dropdown para rol y estado.
- El botón “Add New User” permite abrir el formulario de alta.
- La tabla soporta paginación y muestra el rango de resultados actuales.
- El diseño y experiencia de usuario siguen el mock adjunto (layout, colores, botones, feedback visual).
