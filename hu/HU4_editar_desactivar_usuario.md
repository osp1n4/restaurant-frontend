# HU4: Editar y desactivar usuarios (Frontend)

**Como** Administrador  
**Quiero** poder editar datos de usuarios, restablecer contraseñas y desactivar cuentas  
**Para** mantener actualizada la información y controlar el acceso.

## Criterios de Aceptación
- Desde la tabla/lista de usuarios, hay un botón “Editar” que abre el mismo formulario pre-llenado.
- El formulario permite modificar el nombre y el rol del usuario.
- Hay opción para restablecer la contraseña (nuevo campo o botón).
- El botón para desactivar cuenta está disponible y, al usarlo, cambia el estado del usuario a inactivo (sin eliminarlo).
- El administrador no puede desactivar su propia cuenta (el botón está deshabilitado o no aparece).
- Todos los mensajes de éxito/error y la experiencia visual siguen el estilo del mock.
